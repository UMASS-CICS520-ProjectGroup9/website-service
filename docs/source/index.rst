Academic Ultimate Map Documentation
===================================

Welcome to the documentation for the **Academic Ultimate Map** website service.

Overview
--------
This project is a modular Django web application for academic strategy and success.

Setup & Usage
-------------
1. **Install dependencies**:
   ::

      pip install -r requirements.txt

2. **Run server**:
   ::

      python manage.py runserver
